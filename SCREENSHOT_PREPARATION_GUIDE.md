# Screenshot Preparation Guide - Cyberstition

## Requirements

### Phone Screenshots
- **Quantity**: Minimum 2, maximum 8
- **Aspect Ratio**: 16:9 or 9:16
- **Format**: PNG or JPG
- **Size**: 
  - Phone: 320px to 3840px (width or height)
  - Recommended: 1080x1920 (portrait) or 1920x1080 (landscape)
- **File Size**: Max 8MB per image

### Tablet Screenshots (Optional)
- **Quantity**: 1-8
- **Aspect Ratio**: 16:9 or 9:16
- **Format**: PNG or JPG

### Feature Graphic
- **Size**: 1024x500 pixels
- **Format**: PNG or JPG
- **File Size**: Max 15MB

## Screenshot Best Practices

### Do's ✅
- Use real app screens (not mockups)
- Show actual features working
- Include clear, readable text
- Use consistent styling
- Show different features
- Highlight key benefits
- Use high resolution
- Ensure good contrast

### Don'ts ❌
- Don't use placeholder text
- Don't show errors or crashes
- Don't use low resolution
- Don't include personal information
- Don't show outdated UI
- Don't use misleading content
- Don't include watermarks
- Don't show incomplete features

---

## Text Overlay Guidelines

### Font
- Use clear, readable fonts
- Sans-serif recommended
- Minimum 24px on phone screenshots

### Placement
- Top or bottom of image
- Don't cover important UI elements
- Use semi-transparent background if needed

### Content
- Keep it short (5-7 words)
- Focus on benefits
- Use action words
- Match screenshot content

### Examples
- "Detect scams instantly"
- "100% private analysis"
- "Verify before you trust"
- "AI-powered detection"
- "Export detailed reports"

## Tools for Screenshot Creation

### Android Studio
1. Open app in emulator
2. Extended Controls → Screenshot
3. Save as PNG

### Physical Device
1. Take screenshots normally
2. Transfer to computer
3. Edit if needed

### Editing Tools
- **Free**: GIMP, Paint.NET, Canva
- **Paid**: Photoshop, Sketch, Figma
- **Online**: Canva, Remove.bg

## Quick Start

1. **Prepare app**: Ensure latest version, good sample data
2. **Take screenshots**: Use Android Studio or device
3. **Add overlays**: Use editing tool (optional)
4. **Verify requirements**: Check size, format, count
5. **Organize files**: Name consistently
6. **Upload to Play Console**: Use in store listing
